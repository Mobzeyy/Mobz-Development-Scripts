# Mobz Dependencies Module for FiveM

Mobz is a universal dependency module for **QBCore**, **ESX**, and **Ox Inventory**. It provides:

- Inventory management
- Notifications
- Usable items
- Callbacks for jobs, gangs, money, and items
- Full server/client exports for use in your scripts

---

## **Installation**

1. Place this resource in your server `resources` folder.
2. Add to your `server.cfg`:

```cfg
ensure mobz-dependencies
